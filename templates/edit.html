The format of the access list is as follows:

  • each line describes an RFID token serial number and matching person, separated by a colon (":"): 
    a8948afefef22:Johnny Bigpants
  • lines starting with a pound sign ("#") can be used for comments: 
    # I'm a comment.
  • to allow a user to unlock the doors via logging in, also add: 
    :user=<username>:password=<password>
  • password is a sha1 hash of the password string. You can make one on Linux by running 
    $  echo -n secret_password | sha1sum
    (note the extra space before "echo" - that will keep it out of shell history)
  • to allow a user to add users and edit access list, also add: 
    :admin=yes
